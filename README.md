# TERMUX Parrot Shell
Beautify your Termux 
# Installation
```bash
apt install git -y
cd $HOME
git clone https://github.com/techtricks1/termux-shell.git
cd termux-shell
chmod +x install.sh
./install.sh
```
The termux shell style gets changed!!
# Usage
###### To reinstall after cloning
```bash
techtricks -i
```
###### To restore you old bashrc
```bash
techtricks --restore
```
